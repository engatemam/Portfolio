## ✨ Features <a name="features"></a>
Here's what makes this portfolio special:
- 📱 **Responsive Design**: Beautifully optimized for both desktop and mobile devices.
- 🖱️ **Interactive UI**: Featuring custom cursor elements, preloader, and smooth scroll progress tracking.
- 🎨 **Modern Aesthetics**: Sleek color palette with a stunning "Netflix-style" navigation bar.
- 📚 **Comprehensive Sections**:
  - 🏠 `Home`: A welcoming hero section.
  - 🛤️ `My Journey`: A timeline of my education and experiences.
  - 🛠️ `Skills`: A showcase of my technical and soft skills.
  - 🚀 `Projects`: Highlighting my academic and personal creations.
  - 🎓 `Certificates`: Proud display of my professional milestones.
  - 📩 `Contact`: Easy ways to reach out!
