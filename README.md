# Eng. Ahmed Tarek Emam - Personal Portfolio

Welcome to my personal portfolio website! This repository contains the source code for my interactive portfolio, showcasing my journey, skills, projects, and certificates.

## About Me
I am Ahmed Tarek Emam, a Senior Electrical Engineering Student specializing in Electronics and Communications at Suez Canal University. I am passionate about bridging the gap between theoretical knowledge and practical applications in the engineering field.

## Features
- **Responsive Design**: The website is fully responsive and optimized for both desktop and mobile devices.
- **Interactive UI**: Includes custom cursor elements, preloader, scroll progress bar, and a modern navigation bar.
- **Sections**: 
  - Home: Brief introduction and hero section.
  - My Journey: Timeline of my experience and education.
  - Skills: Highlights of my technical and soft skills.
  - Projects: Showcase of my academic and personal projects.
  - Certificates: Display of my professional certifications.
  - Contact: Ways to get in touch with me.

## Technologies Used
- HTML5
- CSS3 (Vanilla CSS with Custom Variables, Flexbox, CSS Grid)
- JavaScript (Vanilla JS for DOM manipulation, event handling, and interactivity)
- FontAwesome (for icons)
- Google Fonts (Poppins)

## Project Structure
- `index.html`: The main HTML document containing the structure of the portfolio.
- `styles.css`: The stylesheet defining the design, layout, animations, and responsive behavior.
- `script.js`: The JavaScript file handling interactive features like the custom cursor, preloader, scrolling effects, and mobile menu.
- `assets/`: Directory containing images and other media assets used in the project.
- `certificates/`: Directory containing images of certificates.
- `Ahmed_Tarek_Emam_CV.pdf`: My resume/CV.

## How to Run Locally
1. Clone the repository to your local machine.
2. Open the `index.html` file in any modern web browser.
3. No build tools or dependencies are required as this is a static website using vanilla web technologies.
